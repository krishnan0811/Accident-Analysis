# Accident Analysis Dashboard


##  Overview
This project presents an interactive dashboard built using Tableau to analyze road accident data.  
It provides meaningful insights into accident trends, fatalities, vehicle involvement, and environmental factors affecting road safety

##  Objectives
- Analyze accident trends over time  
- Identify high-risk locations  
- Study the impact of weather and road conditions  
- Analyze vehicle types involved in accidents  
- Understand the effect of alcohol on accidents  

---

##  Dataset
The dataset includes detailed accident-related information such as:

- Location (State & City)  
- Date (Year, Month, Day)  
- Time of accident  
- Accident severity  
- Vehicles involved  
- Casualties & fatalities  
- Weather conditions  
- Road type & condition  
- Driver details (age, gender, license status)  
- Alcohol involvement  

---

##  Tools & Technologies
- Tableau  
- Excel / CSV  

---

##  Dashboard Features
-  KPI Cards (Total Accidents, Fatalities, Casualties, Vehicles)  
-  State-wise accident analysis (Map)  
-  Time-based trends (Hourly analysis)  
-  Weather condition analysis  
-  Vehicle type distribution  
-  Road condition insights  
-  Alcohol involvement analysis  

---

##  Key Insights
- Accidents are higher during peak hours  
- Weather conditions like rain increase accident rates  
- Two-wheelers and heavy vehicles are highly involved  
- Alcohol plays a major role in accidents  
- Certain regions show consistently high accident counts  

---

##  Recommendations
- Improve traffic control during peak hours  
- Increase awareness about drunk driving  
- Enhance road infrastructure in high-risk areas  
- Enforce stricter traffic regulations  

---

##  Challenges
- Handling missing data  
- Data cleaning and transformation  
- Designing meaningful visualizations  

---

##  Conclusion
The dashboard helps in understanding accident patterns and supports data-driven decisions to improve road safety.
